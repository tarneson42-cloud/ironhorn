5. App Architecture

IRONHORN is a single HTML file Progressive Web App (PWA). No server required. No framework. No build process. Pure HTML, CSS, and vanilla JavaScript only — built for maximum Safari iOS compatibility.

TabContents
🏠 HomeECG animation, 6 stat tiles, Now card, Next Up reminders, Identity strip, Mission statement
📅 Schedule18-item daily timeline 5:45AM-9:30PM, tap-to-complete pills, progress badge

🏥 Medical8 vital cards, water cups, nutrition rings, Food Tracker (USDA), myAir CPAP, 10 medications

6. Technical Specifications

File typeSingle .html file — no dependencies

14. Safari Compatibility Rules

These patterns crash Safari on iPhone silently. Never use them in IRONHORN JavaScript:

PatternExampleFix
Backtick template literals`Hello ${name}`Use string concatenation
Optional chainingobj?.propertyUse if(obj) checks
Nullish coalescingval ?? defaultUse val || default
Object spread{...obj}Use Object.assign or manual copy
setLineDashctx.setLineDash([4,4])Draw manual dash segments
Math.min.applyMath.min.apply(null,arr)Use for loop
createLinearGradientctx.createLinearGradient()Use flat fill color
innerHTML with onchangediv.innerHTML = ... onchange="fn('key')"Use createElement + data attributes
Regex with backticks/```/gRemove or use string methods
Undefined function callsrMe calls initQuickFoods()Verify every called function exists
