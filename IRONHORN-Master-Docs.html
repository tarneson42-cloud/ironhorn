📱 How to Update IRONHORN

Every time a new version is built — do this:

  • Download IRONHORN.html from Claude
  • Go to github.com/tarneson42-cloud/ironhorn
  • Click Add file → Upload files
  • Upload IRONHORN.html AND IRONHORN-Source-Code.html
  • Click Commit changes
  • Wait 2 minutes — app updates automatically, no reinstall needed
The Home Screen icon auto-updates because it reads from the same URL. IRONHORN-Source-Code.html is the locked backup — upload it every time too.

If Google Script breaks — redeploy:

  • Go to script.google.com → open IRONHORN project
  • Click Run → authorize with tarneson42@gmail.com
  • Deploy → New Deployment → Web app → Execute as Me → Access Anyone
  • Copy new URL → update hardcoded URL in IRONHORN.html → upload to GitHub
